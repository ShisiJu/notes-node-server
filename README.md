# notes-node-server

nodejs 项目用于给前端提供api接口；

技术栈：

* 语言：TypeScript
* web框架：egg 
* ORM框架：Sequelize 
* 打包管理：Webpack 
* 运维：PM2 
* 数据库：postgresql / mongodb / redis
* 测试：mocha

----

数据库应用场景

postgresql ：

* 复杂的业务逻辑
* 重要的数据

mongodb :

* 应用服务器的日志记录
* 存储一些监控数据
* 待完善的新功能

redis ：

* 临时数据
* 缓存

nodejs 后端； 提供api
